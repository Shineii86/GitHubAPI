✨ Powerful Features

Everything you need to analyze developer profiles

Advanced Scoring

8 weighted metrics including stars, repos, activity, languages, followers, account age, contributions, and streak → 0-100 score + game rank (GODLIKE → BEGINNER).

Contribution Streaks

Real current & longest streak data from GitHub GraphQL calendar API. Track developer consistency and activity patterns.

AI Summaries

Optional GPT-4o mini integration for strengths/weaknesses analysis. Get intelligent insights about any developer profile.

Language Insights

Automatic detection of top programming languages across repositories with usage statistics and trends.

Compare Users

Side-by-side comparison of two developers. Perfect for candidate screening and team building decisions.

SVG Badges & Cards

Embeddable SVG badges (no animation) and profile cards with custom backgrounds & theme overlays for GitHub READMEs.

Redis Caching

5-minute intelligent cache to reduce API calls and improve response times. Optional but recommended.

Serverless Ready

Deploy to Vercel in one click. Built for serverless environments with minimal configuration required.

📡 API Endpoints

Simple REST API with powerful responses

GET/api/user/:username

Returns complete profile analysis with score, rank, stats, and AI summary.

bash
$ curl https://githubsmartapi.vercel.app/api/user/shineii86
{
  "username": "shineii86",
  "score": 68,
  "rank": "MASTER",
  "profile": {
    "followers": 5000,
    "publicRepos": 8,
    "accountAgeYears": 12
  },
  "stats": {
    "totalStars": 1200,
    "totalContributions": 243,
    "currentStreak": 2,
    "longestStreak": 7
  },
  "aiSummary": "Moderate activity with room for growth. Good language diversity.",
  "cached": false
}
GET/api/compare/:user1/:user2

Side-by-side comparison of two GitHub users.

GET/api/badge/:username

Returns an SVG badge with rank and level (e.g., "MYTHIC • LV90"). No animation. Supports ?theme=dark|light.

GET/api/card/:username

Generate a beautiful SVG profile card. Supports theme (dark/light), animated (true/false), and bgImage (1–6) for custom backgrounds with theme overlays.
